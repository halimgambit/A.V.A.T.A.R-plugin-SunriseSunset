# SunriseSunset
Vérifiez les heures précises de lever et de coucher du soleil pour n'importe quel jour

- This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework. 

 ## 🎯 Usage
Commandes :
- c'est quand le lever du soleil, le lever du soleil
- c'est quand le cocher du soleil, le coucher du soleil

## Multi-room

The `SunriseSunset` plugin is fully multi-room.

## Multi-language

The `SunriseSunset` plugin relies solely on the system's available languages.
